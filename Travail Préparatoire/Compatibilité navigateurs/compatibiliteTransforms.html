Compatibilité avec les différents navigateurs de la propriété transform

Il est souvent nécessaire d'utiliser les préfixes vendeurs (-webkit-, -moz-, etc.) devant la propriété transform pour utiliser les transformations 2D sur les versions actuelles et passées de navigateurs.

  • -webkit-transform :

    • Safari 3.1+ / Safari Mobile (iOS 3.2)

    • Chrome 4+ / Chrome Mobile (Android 4)

    • Android / Browser 2.1+

  • -moz-transform :

    • Firefox >= 3.5 à <\16

  • -o-transform :

    • Opera >= 10.5 à < 12.1

  • -ms-transform :

    • Internet explorer = 9
